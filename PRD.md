# Double - 产品需求文档 (PRD)

## 产品定位
**K12 教培从业者专用 AI 助理** - 帮助老师高效完成作文批改、资料整理、课件制作和家长沟通。

---

## 1. 项目背景

### 1.1 问题陈述
- K12 教培老师每天花费大量时间批改作文/作业，重复性工作多
- 教学资料分散，收集整理耗时
- 制作课件、教案占用大量备课时间
- 需要频繁回复家长消息，沟通效率低
- 现有工具要么太通用（不够专业），要么太贵（小机构用不起）

### 1.2 解决方案
**Double** 是一款面向 K12 教培从业者的本地优先 AI 助理。

### 1.3 目标用户
- K12 培训机构老师
- 独立教师/家教
- 小型教育机构

---

## 2. 开发计划

### Phase 1: 作文批改 MVP（2 周）
- 文件上传（Word/PDF/TXT）
- AI 批改接口对接
- 评分规则引擎
- 学生档案基础 CRUD
- 批改结果展示界面

### Phase 2: 批量处理 + 资料管理（2 周）
### Phase 3: 课件 + 沟通（2 周）
### Phase 4: 优化发布（1 周）

---

## 3. 技术架构
- Electron 28+
- DeepSeek/Moonshot API
- JSON/JSONL 本地存储
- Puppeteer (资料剪藏)
