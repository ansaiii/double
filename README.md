# Double - Cross-Platform Application

A cross-platform application for Windows, macOS, and Linux.

## Project Setup

Choose your preferred framework:

### Option 1: Electron (JavaScript/TypeScript)
- **Pros**: Mature, large ecosystem, easy web development
- **Cons**: Larger bundle size, higher memory usage
- **Best for**: Complex desktop apps, web developers

### Option 2: Tauri (Rust + Web)
- **Pros**: Small bundle size, fast, secure
- **Cons**: Rust learning curve, newer ecosystem
- **Best for**: Performance-critical apps, security-focused apps

### Option 3: Flutter (Dart)
- **Pros**: Single codebase for mobile/desktop/web, beautiful UI
- **Cons**: Larger runtime, newer desktop support
- **Best for**: Apps that also target mobile platforms

### Option 4: Qt (C++/Python)
- **Pros**: Native performance, mature, professional
- **Cons**: C++ complexity, licensing considerations
- **Best for**: Professional desktop applications

## Next Steps

1. Choose a framework
2. Run the corresponding setup script
3. Start development!
